# braintree-payments-demo
Demo project for Braintree Payments

### Run Server
```
$ cd server
$ pip install -r requirements.txt
$ python manage.py migrate
$ python manage.py runserver
```

### Run Client
```
$ cd client
$ npm install
$ npm start
```
